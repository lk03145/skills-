### Seedance Prompt

```text
电影级写实短剧风格，15秒，9:16竖屏，情绪紧张，节奏清晰，角色外貌和服装保持与参考图一致，画面真实自然，光影有戏剧张力。

0-3秒：远景建立[S01场景]，镜头缓慢推入，环境中出现[关键氛围或冲突前兆]，@图片3作为场景参考。
3-6秒：中景跟拍[C01动作]，人物表情从[情绪A]转为[情绪B]，服装和面部特征参考@图片1。
6-9秒：近景切入[冲突事件]，[C02或P01]进入画面，镜头轻微手持晃动制造紧张感，参考@图片2和@图片4。
9-12秒：特写[C01反应或关键道具]，快速推镜，光线压暗，音乐升高，情绪爆发。
12-15秒：镜头缓慢拉开或定格在[尾帧画面]，留下[悬念点]，为下一集续接。

【声音】环境音：[具体环境音]；音效：[动作音效]；配乐：[悬疑/高燃/虐心/温暖]；台词/旁白：[具体台词]
【参考】@图片1用于C01角色一致性，@图片2用于C02角色一致性，@图片3用于S01场景，@图片4用于P01道具，@图片5用于人物站位关系
【站位】参考@图片5的人物左右关系、前后距离和视线方向，保持角色空间位置一致，不要交换左右位置。
```

## 人物关系站位图引用

当本集有多人对话、谈判、打斗、家庭冲突或群像场面时，在素材上传列表中加入 `REL` 或 `BLK` 图片：

```text
图片N：REL01 人物关系总览，用于理解人物关系
图片N：BLK01 关键场景站位图，用于保持人物左右、前后、视线方向
```

Prompt 中加入：

```text
参考@图片N的站位关系：C01位于画面左侧前景，C02位于画面右侧中景，C03位于背景入口处；三人视线方向和前后距离保持一致，不要交换左右位置。
```

### 尾帧描述

```text
尾帧中，[主体]位于画面[位置]，[表情/动作]停在[状态]，背景是[场景与光线]，[关键道具或人物]清晰可见，情绪停在[悬念/震惊/愤怒/悲伤]，下一集可以从该动作继续。
```

## E02 续接提示

如需延续上一集视频，可在提示词开头加入：

```text
将@视频1延长15秒，保持上一集尾帧的角色位置、服装、光线和情绪，动作自然延续。
```

然后继续写新的 0-15 秒时间轴。

