# AI短剧工作流

用途：默认短剧制作工作流目录。把一个短剧剧本转成可生产的 AI 视频前期包，包含资产提示词、分镜头脚本、Seedance 时间轴提示词、批量生图清单和连续性质检。

## 默认制作规格

| 项目 | 默认值 | 可调整项 |
| --- | --- | --- |
| 画幅 | 9:16 竖屏 | 16:9、2.35:1 |
| 单集时长 | 15 秒 | 5 秒、10 秒、30 秒 |
| 集数 | 5 集 | 按剧本体量调整 |
| 风格 | 电影级写实 | 动画、水墨、赛博、古风、复古 |
| 分镜节奏 | 0-3 / 3-6 / 6-9 / 9-12 / 12-15 秒 | 可按剧情重排 |

## 文件夹说明

| 文件夹 | 用途 | 核心文件 |
| --- | --- | --- |
| `00_原始剧本` | 存放用户给的原剧本、梗概、人物设定 | `剧本输入模板.md` |
| `01_场景人物道具提示词` | 提取人物、场景、道具，并生成图片模型提示词 | `资产提取表.md` |
| `02_剧本分镜头脚本` | 生成镜号、景别、运镜、画面、时长、音效台词 | `分镜头脚本模板.md` |
| `03_Seedance剧本提示词` | 生成 Seedance 2.0 可用的分集提示词 | `Seedance提示词模板.md` |
| `04_连续性与质检` | 检查资产引用、时长、镜头连续、敏感词风险 | `质检清单.md` |
| `05_提示词规则库` | 整合桌面提示词，形成可复用规则库 | `05_总控生产流程.md` |
| `06_本集人物场景道具图` | 根据 C/S/P 提示词批量生成和验收本集参考图 | `批量生图清单.md` |
| `07_人物关系站位图` | 制作人物关系图和关键场景站位图，辅助 Seedance 稳定多人空间关系 | `人物关系站位图清单.md` |
| `99_成片与素材输出` | 存放导出的参考图、生成视频、最终成片 | `README.md` |

## 工作流

1. 把原始短剧剧本放入 `00_原始剧本`。
2. 先做剧情解析：核心钩子、起承转合、人物关系、场景列表、关键道具。
3. 在 `01_场景人物道具提示词` 生成资产表，统一使用：
   - 人物：`C01-C99`
   - 场景：`S01-S99`
   - 道具：`P01-P99`
4. 在 `02_剧本分镜头脚本` 生成分镜表，字段必须包含镜号、景别、运镜、画面描述、分镜时长、音效台词。
5. 在 `03_Seedance剧本提示词` 按集输出 Seedance 时间轴提示词，默认每集 15 秒。
6. 在 `04_连续性与质检` 检查每集是否能接上下一集，尤其是尾帧描述和下一集首帧。
7. 如需要扩写剧本、资产提取、分镜细化或打斗专项，优先调用 `05_提示词规则库` 中对应规则。
8. 在 `06_本集人物场景道具图` 根据资产表批量生成人物、场景、道具参考图，并完成图片验收。
9. 在 `07_人物关系站位图` 生成关系图和站位图，并在 Seedance 提示词中引用，保持多人场景左右、前后、视线方向一致。
