## 文生图提示词公式

```text
[景别/视角]，[角色完整外观描述]，[静态瞬间动作]，[构图手法]，[环境场景和关键道具]，[情绪氛围]，[视觉风格和镜头质感]，[光影效果]，首帧静态画面，角色和道具一致性参考
```

## 图生视频提示词公式

```text
[镜头描述 + 镜头运动速度/幅度]，[主体行为链：动作A->动作B->动作C]，[台词/口型/情绪变化]，[背景变化和环境变化]，[稳定性要求：面部清晰、光线稳定、动作自然、无穿模]
```

## 15 秒节奏模板

| 时间段 | 功能 |
| --- | --- |
| 0-3秒 | 黄金开场，强视觉冲击，直接切入核心冲突 |
| 3-7秒 | 核心叙事，交代人物关系、冲突、线索 |
| 7-12秒 | 深化推进，冲突升级、情绪递进 |
| 12-15秒 | 悬念收尾，反问、未完成动作、突然意外或强特写 |
