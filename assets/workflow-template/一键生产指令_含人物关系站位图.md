# 一键生产指令（含人物关系站位图）

把下面这段复制给 Codex，然后粘贴你的短剧剧本或题材。

```text
请使用 seedance-storyboard-generator 工作流，把以下短剧剧本/题材制作成 AI 短剧前期生产包。

制作规格：
- 画幅：9:16 竖屏
- 默认集数：5 集
- 单集时长：15 秒
- 视觉风格：电影级写实短剧风格
- 输出语言：中文
- 参考规则库：05_提示词规则库

请分别输出并写入项目文件夹：

0. 剧本规范化
   - 如果我给的是题材或梗概，先按 90 秒短剧结构扩写为完整剧本
   - 如果我给的是完整剧本，不改变剧情走向，只做结构分析
   - 剧本必须有强冲突、强情绪、强反转、强悬念
   - 输出到 00_原始剧本

1. 场景、人物、道具提示词提取
   - 人物编号 C01-C99
   - 场景编号 S01-S99
   - 道具编号 P01-P99
   - 每个资产都要有图片生成提示词
   - 人物必须包含姓名、身份/动机、外貌、发型、服饰、气质神态、视觉识别点
   - 场景必须为空场景，无任何人物，并包含环境、光影、天气/时间、氛围
   - 道具必须包含剧情功能、材质/颜色/形状、识别点、出现位置、状态变化
   - 输出到 01_场景人物道具提示词

2. 剧本分镜头脚本
   - 按集输出
   - 包含镜号、时间段、分镜时长、导演定调、景别、拍摄角度、运镜、画面描述、文生图提示词、图生视频提示词、音效台词、出场资产、避坑备注
   - 每集默认 15 秒，优先拆成 0-3、3-6、6-9、9-12、12-15 秒
   - 文生图提示词必须是静态首帧
   - 图生视频提示词必须从首帧延展动作链，并能在对应时长内完成
   - 输出到 02_剧本分镜头脚本

3. 本集人物、场景、道具图批量生成
   - 根据 01_场景人物道具提示词 中的 C/S/P 资产提示词，生成批量生图清单
   - 人物图放入 06_本集人物场景道具图/人物_C
   - 场景图放入 06_本集人物场景道具图/场景_S
   - 道具图放入 06_本集人物场景道具图/道具_P
   - 待生成任务放入 06_本集人物场景道具图/待生成_Queue
   - 合格图片复制到 06_本集人物场景道具图/已完成_Done
   - 不合格图片放入 06_本集人物场景道具图/废稿_Rejects
   - 同时输出图片验收表
   - 输出到 06_本集人物场景道具图

4. 人物关系站位图
   - 根据人物资产、场景资产和分镜脚本，生成人物关系图和关键场景站位图
   - 关系图放入 07_人物关系站位图/关系图_Relations
   - 站位图放入 07_人物关系站位图/站位图_Blocking
   - 最终用于 Seedance 的参考图放入 07_人物关系站位图/Seedance参考图_References
   - 待生成任务放入 07_人物关系站位图/待生成_Queue
   - 合格图片复制到 07_人物关系站位图/已完成_Done
   - 不合格图片放入 07_人物关系站位图/废稿_Rejects
   - 站位图必须标注角色ID、姓名、左右位置、前后距离、视线方向、镜头方向
   - 输出到 07_人物关系站位图

5. Seedance 剧本提示词
   - 每集包含素材上传列表
   - 每集包含可直接复制进 Seedance 的 Prompt
   - 每集包含声音设计、参考素材说明、尾帧描述
   - 第二集起如需续接上一集，请写明“将@视频1延长15秒”
   - 提示词中必须写明角色、服装、道具、场景与参考素材保持一致
   - 多人场景必须写明是否引用 07_人物关系站位图 中的 REL/BLK 参考图
   - 输出到 03_Seedance剧本提示词

6. 打斗动态提示词
   - 如果剧本包含打斗、追逐、法术碰撞或动作反击，额外输出 15 秒打斗动态提示词
   - 按 0-3、3-6、6-9、9-12、12-15 秒五个节点输出
   - 每个节点包含动作、特效、运镜、光影、人物情绪
   - 尾帧明确胜负、落点或下一集悬念
   - 输出到 03_Seedance剧本提示词

7. 连续性与质检
   - 检查所有资产引用是否匹配
   - 检查每集是否覆盖完整时长
   - 检查尾帧和下一集首帧是否能自然衔接
   - 检查角色外观、服装、道具、场景光影是否一致
   - 检查 06_本集人物场景道具图 中的图片是否命名规范、是否已完成验收
   - 检查 07_人物关系站位图 中的关系图/站位图是否已在 Seedance 提示词中正确引用
   - 标出可能的敏感词风险，并给出替代表达
   - 输出到 04_连续性与质检

项目名：
[填写项目名]

原始剧本或题材如下：

[在这里粘贴剧本或题材]
```
