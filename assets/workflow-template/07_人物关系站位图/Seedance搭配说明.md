# Seedance 搭配说明

人物关系站位图用于解决 Seedance 生成视频时常见的三个问题：

1. 人物左右位置跳变。
2. 对话对象或视线方向混乱。
3. 多人场景中前后关系、主次关系不稳定。

## 上传建议

Seedance 参考图数量有限时，优先级如下：

| 优先级 | 素材类型 | 说明 |
| --- | --- | --- |
| 1 | C人物图 | 主角、对手、关键角色一致性 |
| 2 | S场景图 | 主要空间和光影一致性 |
| 3 | BLK站位图 | 多人对话、冲突、谈判、打斗前后位置 |
| 4 | P道具图 | 关键道具或反转线索 |
| 5 | REL关系图 | 剧情关系复杂时使用 |

## 素材上传列表写法

```markdown
| 上传位置 | 资产ID | 素材描述 | 用途 |
| --- | --- | --- | --- |
| 图片1 | C01 | 主角一致性参考 | 人物外貌、服装、气质 |
| 图片2 | C02 | 对手一致性参考 | 人物外貌、压迫感 |
| 图片3 | S01 | 主场景参考 | 空间、光线、氛围 |
| 图片4 | BLK01 | 关键场景站位图 | 左右位置、前后关系、视线方向 |
```

## Prompt 搭配写法

```text
参考@图片4的站位关系：C01位于画面左侧前景，C02位于画面右侧中景，C03位于背景入口处；三人视线方向和前后距离保持一致，不要交换左右位置。
```

多人对话场景：

```text
根据@图片4保持三人站位，C01面对C02说话，C03在后方观察；镜头从C01侧后方推近，人物左右关系不变。
```

谈判/酒局/会议场景：

```text
参考@图片4的桌面座位关系，C01坐在画面左侧，C02坐在画面右侧，C03站在后方；镜头切换时保持座位顺序和视线方向一致。
```

打斗/追逐场景：

```text
参考@图片4的起始站位，C01从左前方冲向C02，C02位于右后方后退，镜头跟随C01移动，保持角色方向不反转。
```

## 验收检查

| 检查项 | 要求 |
| --- | --- |
| 角色ID | 每个人物必须标注 C 编号 |
| 左右关系 | 画面左/右必须明确 |
| 前后关系 | 前景/中景/背景必须明确 |
| 视线方向 | 谁看谁要明确 |
| 镜头方向 | 从哪一侧拍摄要明确 |
| Seedance引用 | Prompt 中必须写明参考哪张图片 |
