# 人物关系站位图

本文件夹用于制作和存放“人物关系图”和“人物站位图”，供 Seedance 生成视频时作为参考图使用，并与 `03_Seedance剧本提示词` 中的素材上传列表、分镜提示词搭配。

## 文件夹结构

```text
07_人物关系站位图/
  关系图_Relations/
  站位图_Blocking/
  Seedance参考图_References/
  待生成_Queue/
  已完成_Done/
  废稿_Rejects/
  README.md
  人物关系站位图清单.md
  Seedance搭配说明.md
```

## 子文件夹用途

| 文件夹 | 用途 |
| --- | --- |
| `关系图_Relations` | 存放人物关系图，如亲属、敌对、合作、暧昧、雇佣等 |
| `站位图_Blocking` | 存放关键场景的人物站位图，标注左右、前后、远近、视线方向 |
| `Seedance参考图_References` | 存放最终用于 Seedance 上传的关系/站位参考图 |
| `待生成_Queue` | 放待生成的关系图、站位图任务 |
| `已完成_Done` | 放通过验收、可直接上传 Seedance 的参考图 |
| `废稿_Rejects` | 放错误站位、关系不清、视觉混乱的废稿 |

## 推荐命名

```text
[项目名]_E[集数]_REL01_人物关系总览.png
[项目名]_E[集数]_BLK01_[场景名]_站位图.png
[项目名]_E[集数]_REF01_Seedance站位参考.png
```

示例：

```text
重生在1999_E08_REL01_人物关系总览.png
重生在1999_E08_BLK01_酒吧卡座_陆豪周成钱峰站位.png
重生在1999_E08_REF01_Seedance上传站位参考.png
```

## 制作原则

1. 人物关系图用于理解人物之间的关系，不追求电影画面美感，优先清晰。
2. 人物站位图用于 Seedance 参考，必须明确谁在左、谁在右、谁在前景、谁在背景。
3. 站位图要标注视线方向、运动方向、镜头方向，避免视频生成时人物位置跳变。
4. 同一场景多段视频使用同一站位图，可提高角色空间连续性。
5. 关系/站位图只做参考，不要替代人物 C 图、场景 S 图、道具 P 图。

## 与 Seedance 搭配

在 `03_Seedance剧本提示词` 的素材上传列表中加入：

```text
图片N：REL01 或 BLK01，用于人物关系和站位参考
```

在 Prompt 中写明：

```text
参考@图片N的人物左右关系、前后站位和视线方向，保持角色空间位置一致。
```
