# 本集人物、场景、道具图

本文件夹是通用工作流中的“批量生图阶段”，用于根据 `01_场景人物道具提示词/资产提取表.md` 中的 C/S/P 提示词批量生成图片，并把合格图片整理成 Seedance 上传素材。

## 文件夹结构

```text
06_本集人物场景道具图/
  人物_C/
  场景_S/
  道具_P/
  待生成_Queue/
  已完成_Done/
  废稿_Rejects/
  README.md
  批量生图清单.md
  图片验收表.md
```

## 子文件夹用途

| 文件夹 | 用途 |
| --- | --- |
| `人物_C` | 存放本集人物参考图，如 C01、C02 |
| `场景_S` | 存放本集空场景参考图，如 S01、S02 |
| `道具_P` | 存放本集关键道具参考图，如 P01、P02 |
| `待生成_Queue` | 放待生成提示词、临时批量任务、未完成图片 |
| `已完成_Done` | 放已经通过验收、可上传 Seedance 的图片 |
| `废稿_Rejects` | 放不合格图片，保留用于复盘问题 |

## 推荐命名

```text
[项目名]_E[集数]_C01_角色名.png
[项目名]_E[集数]_S01_场景名.png
[项目名]_E[集数]_P01_道具名.png
```

示例：

```text
重生在1999_E08_C01_陆豪.png
重生在1999_E08_S01_酒吧卡座.png
重生在1999_E08_P01_洋酒瓶.png
```

## 使用顺序

1. 先完成 `01_场景人物道具提示词/资产提取表.md`。
2. 把资产表里的 C/S/P 提示词复制到 `批量生图清单.md`。
3. 按 C、S、P 分别生成图片。
4. 把合格图片放入 `人物_C`、`场景_S`、`道具_P`。
5. 同步复制最终可用图片到 `已完成_Done`。
6. 按 `图片验收表.md` 检查角色一致性、场景空镜、道具清晰度。
7. 在 `03_Seedance剧本提示词` 中按上传顺序引用这些图片。
