# 总控生产流程

本文件把桌面提示词整合为可执行的短剧生产流水线。

## 阶段 0：项目初始化

输入：

- 项目名
- 原始剧本、题材或梗概
- 目标集数与单集时长
- 画幅
- 视觉风格
- 是否包含打斗/特效/真实产品/已有参考图

默认值：

```text
目标集数：5集
单集时长：15秒
画幅：9:16竖屏
视觉风格：电影级写实短剧风格
```

## 阶段 1：剧本规范化

如果用户给的是题材或梗概，先按 `01_剧本创作规则.md` 扩写成 90 秒或指定时长剧本。

如果用户给的是完整剧本，只做结构理解，不改剧情走向。

输出到：

```text
00_原始剧本/[项目名]_规范化剧本.md
```

## 阶段 2：资产提取

按 `02_资产提取规则.md` 输出：

- `C` 人物资产
- `S` 空场景资产
- `P` 道具资产
- 统一视觉风格前缀
- 生图提示词
- Seedance 上传素材优先级

输出到：

```text
01_场景人物道具提示词/[项目名]_资产提取表.md
```

## 阶段 3：分镜脚本

按 `03_分镜与视频提示词规则.md` 输出：

- 镜号
- 时间段
- 景别
- 拍摄角度
- 运镜
- 导演定调
- 画面描述
- 文生图提示词
- 图生视频提示词
- 声音/台词
- 避坑备注

输出到：

```text
02_剧本分镜头脚本/[项目名]_E01_分镜头脚本.md
```

## 阶段 4：批量生成本集参考图

按 `06_本集人物场景道具图/批量生图清单.md` 输出：

- 人物图生成任务
- 空场景图生成任务
- 道具图生成任务
- 推荐文件名
- 生成状态
- 验收备注

图片存放到：

```text
06_本集人物场景道具图/人物_C/
06_本集人物场景道具图/场景_S/
06_本集人物场景道具图/道具_P/
```

验收文件：

```text
06_本集人物场景道具图/图片验收表.md
```

## 阶段 5：人物关系站位图

按 `07_人物关系站位图/人物关系站位图清单.md` 输出：

- 人物关系总览图任务
- 本集冲突/合作/亲密关系图任务
- 关键场景站位图任务
- Seedance 上传参考图候选
- 站位图引用说明

图片存放到：

```text
07_人物关系站位图/关系图_Relations/
07_人物关系站位图/站位图_Blocking/
07_人物关系站位图/Seedance参考图_References/
```

搭配说明：

```text
07_人物关系站位图/Seedance搭配说明.md
```

## 阶段 6：Seedance 提示词

按 `03_Seedance剧本提示词/Seedance提示词模板.md` 输出：

- 素材上传列表
- 可复制 Seedance Prompt
- 0-15 秒时间轴
- 声音设计
- 参考素材说明
- 尾帧描述
- 续接下一集的说明
- 人物关系图/站位图引用说明

输出到：

```text
03_Seedance剧本提示词/[项目名]_E01_Seedance提示词.md
```

## 阶段 7：打斗专项

如果剧本含打斗或追逐，按 `04_打斗动态规则.md` 额外生成：

```text
03_Seedance剧本提示词/[项目名]_E01_打斗动态提示词.md
```

## 阶段 8：连续性与质检

按 `04_连续性与质检/质检清单.md` 检查：

- 资产 ID 是否全部映射
- 时间轴是否覆盖完整
- 角色外观、服装、道具是否一致
- 场景光影与空间是否连续
- 多人场景的关系图/站位图是否被正确引用
- 尾帧和下一集首帧是否自然衔接
- 是否存在敏感词或生成风险

输出到：

```text
04_连续性与质检/[项目名]_连续性质检.md
```

## 总控指令

```text
请使用 AI短剧 Seedance 工作流处理以下剧本/题材。

执行顺序：
1. 若是题材或梗概，先扩写为指定时长的短剧剧本；若是完整剧本，不改变剧情走向。
2. 提取 C 人物、S 空场景、P 道具，并生成生图提示词。
3. 生成按集拆分的分镜头脚本，包含导演定调、景别、角度、运镜、画面、文生图提示词、图生视频提示词、声音台词。
4. 根据 C/S/P 资产提示词生成批量生图清单，并把合格图片整理到 06_本集人物场景道具图。
5. 生成人物关系图和关键场景站位图，并整理到 07_人物关系站位图。
6. 生成 Seedance 可复制提示词，每集包含素材上传列表、时间轴、声音设计、参考说明、尾帧描述，并引用关系/站位图。
7. 如含打斗，额外生成 15 秒打斗动态提示词。
8. 最后输出连续性与质检报告。

默认规格：
- 5集
- 每集15秒
- 9:16竖屏
- 电影级写实短剧风格

项目名：
[填写]

剧本/题材：
[粘贴]
```
