# 打斗动态规则

来源整合：`打斗动态提示词提取指令模板.txt`。

## 适用场景

当剧本包含对打、追杀、法术碰撞、玄幻战斗、动作反击等桥段时，使用本规则生成 15 秒动作视频提示词。

## 默认规格

| 项目 | 默认值 |
| --- | --- |
| 风格 | 电影级 3D 玄幻动画风格 |
| 时长 | 15 秒 |
| 节点 | 每 3 秒一个节点，共 5 个 |
| 节奏 | 超高速、紧凑、强视觉冲击 |
| 结局 | 按剧情设定明确胜负或悬念 |

## 15 秒节点

| 时间段 | 功能 | 动作重点 | 特效/运镜 |
| --- | --- | --- | --- |
| 0-3s | 打斗开场 | 快速切入，第一轮攻击或闪避 | 虚影、粒子光效、快速跟拍 |
| 3-6s | 打斗升级 | 连续出拳、踢腿、格挡、反击 | 环绕运镜、冲击波、光影爆发 |
| 6-9s | 白热化 | 高密度动作碰撞，关键招式 | 慢动作卡点、环境碎裂、强特效 |
| 9-12s | 胜负铺垫 | 一方露出破绽或被压制 | 聚焦核心区域，人物情绪分化 |
| 12-15s | 结局/悬念 | 落败、击飞、倒地、定格或反杀前一刻 | 慢动作收束，特效余韵，尾帧清晰 |

## 必填要素

| 要素 | 要求 |
| --- | --- |
| 人物动作 | 快速、夸张、紧凑；动作贴合人物能力和武器 |
| 情绪变化 | 初期紧绷，中期激昂，落败方狼狈，优势方掌控全局 |
| 虚影特效 | 贴合高速动作轨迹，增强速度感 |
| 环境特效 | 地面碎裂、石块飞溅、气流涌动、障碍物破坏 |
| 粒子光效 | 环绕人物和打斗区域，随动作联动 |
| 冲击波 | 碰撞和发力时自然扩散，光影联动 |
| 运镜 | 丝滑跟拍、环绕、快速推拉、急转镜 |
| 画质 | 高帧率、流畅丝滑、无卡顿、无模糊、无噪点 |

## 打斗提示词模板

```text
电影级3D玄幻动画风格，15秒，9:16竖屏，高帧率，流畅丝滑，无卡顿，无模糊。图片1人物与图片2人物进行激烈打斗，动作紧凑，节奏拉满，特效与动作、光影、环境自然联动。

【0-3s】图片1人物与图片2人物快速切入战斗，[具体攻击/闪避动作]，虚影特效贴合动作轨迹，粒子光效环绕，快速跟拍，人物眼神凌厉，光影明亮，直接抓住视觉注意力。

【3-6s】打斗升级，[连续动作链]，加入[巨龙/法相/能量光效，如适用]，冲击波轻微扩散，环绕运镜，人物情绪激昂，光影随特效爆发变化。

【6-9s】打斗白热化，[关键碰撞动作]，环境特效与打斗动作联动，地面碎裂/碎石飞溅/气流涌动，慢动作卡点呈现碰撞瞬间，史诗级光影拉满。

【9-12s】胜负铺垫，[落败方动作迟缓或破绽出现]，[优势方持续压制]，丝滑跟拍，人物情绪分化，光影聚焦核心打斗区域。

【12-15s】结局呈现，[落败方被击飞/倒地/后退/被压制]，[优势方定格站立或保持攻击姿态]，特效逐渐减弱但保留余韵，慢动作卡点，尾帧清晰可续接下一段。
```
