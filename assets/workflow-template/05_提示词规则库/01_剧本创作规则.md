# 剧本创作规则

来源整合：`写剧本指令模板.txt`、`模板提示词框架.docx`。

## 编剧定位

以专业短视频短剧编剧身份创作，目标是高密度、强节奏、强情绪、强反转、画面感强，并适配 AI 视频生成。

## 适用场景

当用户提供的是题材、梗概、小说片段或不完整剧情时，先用本规则扩写成可分镜的短剧剧本。

## 默认剧本规格

| 项目 | 默认值 |
| --- | --- |
| 单集时长 | 90 秒 |
| 人物数量 | 2-4 个主要人物 |
| 场景数量 | 场景简洁，但细节完整 |
| 叙事风格 | 强冲突、强反转、强悬念 |
| 台词风格 | 自然、饱满、有情绪、有性格 |

## 90 秒结构

| 时间段 | 功能 | 要求 |
| --- | --- | --- |
| 0-15 秒 | 开场强冲击 | 直接切入冲突，不铺垫、不废话 |
| 15-60 秒 | 核心冲突推进 | 铺冲突、埋线索、展人物、堆情绪 |
| 60-80 秒 | 反转爆发 | 有伏笔、有逻辑、有打脸感或逆袭感 |
| 80-90 秒 | 结尾强悬念 | 用一句话、一个动作或一个眼神制造追更钩子 |

## 创作硬规则

1. 每一秒都要推动剧情，避免空镜、废话、拖沓。
2. 每一句台词必须服务于冲突、反转、爽点或人物关系。
3. 每个动作必须带情绪、力度和神态，避免只写“走过来”“看着他”。
4. 场景必须写清氛围、光线、色调、环境细节，不只写地点名。
5. 反转应在 65-80 秒附近出现，并在前文埋下伏笔。
6. 结尾必须留下强悬念，不能平淡收尾。

## 剧本输出格式

```markdown
# 短剧剧本《项目名》（第1集）

单集时长：90秒
适配：AI 视频生成 / Seedance / 即梦

## 人物介绍

- 角色名：身份 + 性格 + 核心动机 + 核心立场

## 剧情正文

【0-15秒】场景：地点 + 氛围 + 光线 + 环境细节
人物动作：动作 + 情绪 + 力度 + 神态
角色名（语气 / 神态 / 微表情）：台词

【15-60秒】场景：...
人物动作：...
角色名（语气 / 神态 / 微表情）：台词

【60-80秒】场景：...
人物动作：...
角色名（语气 / 神态 / 微表情）：台词

【80-90秒】场景：...
人物动作：...
角色名（语气 / 神态 / 微表情）：台词
```

## 给 Codex 的可复制指令

```text
请以专业短视频短剧编剧身份，根据我提供的题材或梗概，创作一集 90 秒 AI 短剧剧本。

要求：
- 高密度细节、强冲突、强情绪、强反转。
- 人物 2-4 个，性格鲜明，有动机，有立场。
- 场景简洁但细节完整，写清氛围、光线、色调、环境。
- 动作必须带情绪、力度、神态。
- 台词自然、有性格、有目的，服务冲突、反转或爽点。
- 0-15 秒强冲击开场，15-60 秒冲突推进，60-80 秒反转爆发，80-90 秒强悬念收尾。
- 仅输出最终剧本，不解释规则。

用户题材：
[粘贴题材或梗概]
```
