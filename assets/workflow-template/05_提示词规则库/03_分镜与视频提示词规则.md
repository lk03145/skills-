# 分镜与视频提示词规则

来源整合：`剧本拆分自动分幕指令动模板.txt`、`分镜提示框架.txt`、`模板提示词框架.docx`。

## 分镜总原则

1. 先理解故事，再做导演定调，再拆分镜头。
2. 每个镜头只表达一个核心叙事功能和一个核心情绪。
3. 单镜头建议 1-3 秒，复杂动作最长不超过 5 秒。
4. 每 15 秒片段必须有开场冲击、核心推进、悬念收尾。
5. 分镜必须同时服务文生图首帧和图生视频动作延展。

## 15 秒片段结构

| 时间段 | 功能 | 要求 |
| --- | --- | --- |
| 0-3s | 黄金开场 | 强视觉冲击，直接切核心冲突或关键画面 |
| 3-7s | 核心叙事 | 传递人物关系、冲突、关键线索 |
| 7-12s | 深化推进 | 推进冲突、补充细节、加重情绪 |
| 12-15s | 悬念收尾 | 反问、未完成动作、突然意外、强特写卡点 |

Seedance 版本也可以使用 5 段式：

```text
0-3秒 / 3-6秒 / 6-9秒 / 9-12秒 / 12-15秒
```

## 分镜字段

| 字段 | 要求 |
| --- | --- |
| 镜号 | 例如 E01-SH01 |
| 时间段 | 精确到秒 |
| 时长 | 单镜建议 1-3 秒 |
| 导演定调 | 叙事功能 + 核心情绪 + 光影基调 |
| 景别 | 远景、全景、中景、近景、特写、大特写 |
| 拍摄角度 | 俯拍、仰拍、平拍、侧拍、斜拍、顶拍 |
| 运镜 | 推镜、拉镜、跟镜、摇镜、旋转、固定、手持 |
| 画面描述 | 场景 + 动作 + 情绪 + 光影 + 道具 |
| 文生图提示词 | 静态首帧，不能写持续动作 |
| 图生视频提示词 | 从首帧延展的动作链 |
| 声音/台词 | 环境音、音效、配乐、台词、旁白、字幕 |
| 避坑备注 | 一致性、穿模、比例、光影、口型等 |

## 文生图提示词公式

```text
[景别/视角]，[角色完整外观描述]，[静态瞬间动作]，[构图手法]，[环境场景和关键道具]，[情绪氛围]，[视觉风格和镜头质感]，[光影效果]，首帧静态画面，角色和道具一致性参考
```

要求：

1. 角色跨镜头必须重复完整外观描述。
2. 首帧必须是静态瞬间，不写“边走边说”这类持续动作。
3. 如果后续要露脸，首帧就应露脸；否则使用背影、手部、道具特写。
4. 道具位置、朝向、房间布局要与前后镜头一致。
5. 过细描述容易出错时，用更稳定的宽泛描述。

## 图生视频提示词公式

```text
[镜头描述 + 镜头运动速度/幅度]，[主体行为链：动作A→动作B→动作C]，[台词/口型/情绪变化]，[背景变化和环境变化]，[稳定性要求：面部清晰、光线稳定、动作自然、无穿模]
```

要求：

1. 动作链能在该镜头时长内完成。
2. 用“→ / 立刻 / 紧接着”串联动作，因果清晰。
3. 首帧承接文生图，人物、道具、场景一致。
4. 关键要求放在结尾，例如“保持角色面部清晰、光线稳定”。
5. 台词保持原文，不翻译、不改写；必要时标注语速。

## 导演定调要素库

| 类型 | 可用词 |
| --- | --- |
| 叙事功能 | 交代环境、推动情节、塑造人物、情绪递进、冲突升级、伏笔铺垫、收尾强化、细节强调、时空转换 |
| 核心情绪 | 急切、冰冷、惊恐、茫然、暴怒、平和、焦虑、惋惜、无助、决绝、戏谑、凝重、委屈、狡黠、失落、欣慰 |
| 光影基调 | 低饱和暖调、冷白调、自然柔光、侧逆光、明暗对比、顶光压迫、霓虹冷暖对比、烛光摇曳 |

## 分镜师落地要素库

| 类型 | 可用词 |
| --- | --- |
| 景别 | 远景、全景、中景、近景、双人中景、背影近景、特写、大特写 |
| 构图 | 中心构图、三分构图、框架构图、对角构图、留白构图、水平构图、焦外虚化构图 |
| 镜头质感 | 浅景深、深景深、焦外虚化、清晰全画、胶片质感、高清通透 |
| 运镜 | 固定镜头、缓慢推近、快速推近、慢速横摇、小幅度仰摇、平稳跟拍、轻微手持晃动、快速甩摇 |
| 避坑 | 人物比例正常、肢体动作自然、道具不穿模、光影不杂乱、口型与台词匹配、跨镜头空间统一 |

## 给 Codex 的可复制指令

```text
请把以下短剧剧本拆成 15 秒片段，并生成分镜头脚本和 AI 视频提示词。

要求：
- 每个 15 秒片段按 0-3s、3-7s、7-12s、12-15s 拆分；Seedance 版可再细化为 0-3、3-6、6-9、9-12、12-15。
- 每个镜头包含镜号、时间段、时长、导演定调、景别、拍摄角度、运镜、画面描述、文生图提示词、图生视频提示词、声音/台词。
- 单镜头建议 1-3 秒，复杂动作最长不超过 5 秒。
- 画面描述必须包含场景、动作、情绪、光影、道具。
- 台词保持原文，紧跟对应时间段。
- 检查角色、道具、空间、光影的连续性。

剧本：
[粘贴剧本]
```
