# 资产提取规则

来源整合：`角色设计指令模板.txt`、`场景提取指令模板.txt`、`chatgpt场景提取指令模板.txt`。

## 资产编号

| 类型 | 编号 | 用途 |
| --- | --- | --- |
| 人物 | C01-C99 | 角色一致性参考图、Seedance 人物引用 |
| 场景 | S01-S99 | 空场景图、场景一致性参考 |
| 道具 | P01-P99 | 关键证据、武器、信物、反转线索 |

## 人物提取规则

仅提取主要人物：剧情核心角色、贯穿剧本或在关键剧情中起决定作用的角色。路人和一次性配角不进入主资产，除非需要生成参考图。

每个主要人物必须包含：

| 字段 | 要求 |
| --- | --- |
| 姓名 | 标准姓名，别名/昵称一并标注 |
| 身份 | 职业、地位、剧情定位、核心动机 |
| 外貌 | 肤色、五官、脸型、眉眼鼻唇、面部细节 |
| 发型 | 发型、发色、发丝状态、发饰 |
| 服饰 | 款式、颜色、层次、纹样、材质、配饰、鞋子 |
| 气质神态 | 眼神、气场、日常神态、肢体感 |
| 识别标志 | 标志性颜色、饰品、伤痕、道具、轮廓 |

### 人物生图提示词公式

```text
[统一视觉风格]，[角色姓名]，[年龄与身份]，[身高体型]，[发型发色和发丝质感]，[五官与脸型]，[服饰款式、颜色、层次、纹样、材质、配饰、鞋子]，[气质神态和眼神气场]，[识别标志]，面对镜头垂手而立，站姿标准，全身角色设定图，纯色或简洁背景，8K超高清，细节清晰，光影自然，角色一致性参考
```

## 场景提取规则

场景资产优先提取空场景，不包含人物、身影、衣物碎片或人物动作。

同一地点不同时间、天气、状态必须拆成不同场景。例如：`S01-夜晚客厅` 和 `S02-清晨客厅` 分开。

每个场景必须包含：

| 字段 | 要求 |
| --- | --- |
| 场景名称 | 源于剧本，简洁明确 |
| 环境描写 | 地点、建筑、陈设、空间层次、环境细节 |
| 光影描写 | 光线类型、方向、强弱、明暗关系 |
| 天气/时间 | 明确天气和时间；未写明则按氛围合理补全 |
| 氛围 | 紧张、压抑、温馨、清冷、危险、诡谲等 |
| 空场景标注 | 明确写“空场景，无任何人物” |

### 场景生图提示词公式

```text
[统一视觉风格]，[场景名称]，空场景，无任何人物，[地点与空间结构]，[建筑与陈设细节]，[天气与时间]，[光线方向与明暗层次]，[色调与氛围]，深景深，画面干净，适合作为 AI 视频场景一致性参考
```

## 道具提取规则

提取会推动剧情、承载线索或影响动作的物件。

每个道具必须包含：

| 字段 | 要求 |
| --- | --- |
| 道具名称 | 剧本中的标准名称 |
| 剧情功能 | 证据、信物、武器、药物、钥匙、合同、手机等 |
| 视觉识别点 | 颜色、材质、形状、磨损、文字、符号 |
| 出现场次 | 第几集/第几镜/哪个时间段 |
| 状态变化 | 完好、破损、染尘、发光、被打开等 |

### 道具生图提示词公式

```text
[统一视觉风格]，[道具名称]，[材质、颜色、形状、尺寸]，[关键符号或文字]，[磨损与状态变化]，[放置环境]，特写构图，主体清晰，背景简洁，适合作为关键道具参考图
```

## 给 Codex 的可复制指令

```text
请从以下短剧剧本中提取人物、场景、道具资产，并生成生图提示词。

要求：
- 人物编号 C01-C99，仅提取主要人物。
- 场景编号 S01-S99，仅提取空场景，无任何人物。
- 道具编号 P01-P99，仅提取推动剧情或形成线索的关键物件。
- 每个资产都要有完整描述和可直接用于生图的提示词。
- 使用统一视觉风格前缀，保证后续视频一致性。
- 输出为表格，并附 Seedance 上传素材优先级。

剧本：
[粘贴剧本]
```
