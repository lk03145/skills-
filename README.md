# Seedance Storyboard Generator

Codex skill for turning Chinese stories, scripts, outlines, and short-drama episodes into Seedance-ready AI video production packages.

This GitHub-ready version includes:

- Seedance 2.0 prompt references
- short-drama script adaptation rules
- character, scene, and prop prompt extraction
- storyboard script templates
- Seedance time-axis prompt templates
- batch image-generation queues
- character relationship and blocking reference folders
- continuity and quality-check templates

The reusable project template lives in:

```text
assets/workflow-template/
```

For each new episode, copy `assets/workflow-template` into a project folder such as `项目名_第8集_Seedance生产包`, then place the source script in `00_原始剧本`.
